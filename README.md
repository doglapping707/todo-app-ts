## インストール・起動方法
```
git clone https://github.com/doglapping707/todo-app-ts
cd todo-app-ts
npm install
npm run dev
```

## .envの作成
`.env.local`を作成し、下記をコピーしてください。
```
VITE_API_URL={使用するタスクAPIのURL}
```
